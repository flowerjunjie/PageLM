# 灵犀AI学习系统 2.0 升级计划

## 🎯 核心目标
从"通用AI对话工具"升级为"完整AI学习系统"，与豆包等通用大模型形成差异化定位。

## 📋 升级概览

| Phase | 模块 | 优先级 | 工时 | 状态 |
|-------|------|--------|------|------|
| 1 | 首页界面重构 - 学习模式选择 | High | 8h | 🟡 待开始 |
| 2 | 学习资料强制生成系统 | High | 12h | 🟡 待开始 |
| 3 | 学习档案与知识地图 | Medium | 16h | 🟡 待开始 |
| 4 | 艾宾浩斯复习提醒系统 | Medium | 14h | 🟡 待开始 |
| 5 | 学习周报系统 | Medium | 10h | 🟡 待开始 |
| 6 | Planner作业规划深度集成 | Low | 12h | 🟡 待开始 |

**总计：约72工时（9个工作日）**

---

## 🔍 差异化定位策略

### 对比：灵犀 vs 豆包

| 维度 | 豆包/通用AI | 灵犀（学习系统） |
|------|-----------|----------------|
| **使用目的** | 问什么答什么 | 主动构建知识体系 |
| **数据沉淀** | 聊完即走 | 笔记、卡片、测验长期保存 |
| **学习路径** | 单次问答 | 预习→学习→复习→测验闭环 |
| **家长管控** | 无 | 学习报告、进度追踪 |

### Phase 2: 学习资料强制生成（第2-4天）

**目标**：确保每次AI对话都有学习成果沉淀

**核心逻辑**：
```javascript
// 每次AI回答后自动执行
async function generateLearningMaterials(question, answer) {
  const flashcards = await generateFlashcards(answer);  // 3-5张卡片
  const notes = await generateNotesSummary(answer);      // 笔记摘要
  const quiz = await generateQuizQuestions(answer);      // 3道测验

  // 保存到用户学习档案
  await saveToUserProfile(userId, { flashcards, notes, quiz });
}
```

**前端展示**：
```
AI回答内容...
---
📚 已为你生成学习资料：
   • 3张记忆卡片 [查看] [开始复习]
   • 1份笔记摘要 [查看] [导出]
   • 3道测验题 [测试一下]
```

### Phase 4: 遗忘曲线复习系统（第6-8天）

**目标**：基于科学记忆曲线提升长期记忆效果

**艾宾浩斯遗忘曲线复习点**：
- 20分钟后（首次复习）
- 1小时后
- 9小时后
- 1天后
- 2天后
- 6天后
- 31天后

**SM-2算法**：
```javascript
function sm2(interval, repetition, easiness, quality) {
  // quality: 0-5 (0=完全忘记, 5=完全记住)
  // 根据记忆质量调整下次复习时间
}
```

**复习界面**：
- 类似Anki的卡片翻转
- 自评记忆程度（记得/模糊/忘记）
- 系统根据评分调整下次复习时间

### Phase 6: Planner作业规划增强（第10-12天）

**目标**：从作业管理切入，成为学习流程的核心

**作业拍照识别流程**：
```
1. 拍照上传作业
   ↓
2. OCR识别文字
   ↓
3. AI分解任务步骤
   ↓
4. 生成学习计划
   ↓
5. 定时提醒执行
```

**智能规划示例**：
```
作业：完成数学第5章习题 + 英语作文

AI规划：
📅 今天 19:00-20:00 - 数学习题（预估1小时）
📅 明天 19:00-19:30 - 数学错题整理（30分钟）
📅 明天 19:30-20:30 - 英语作文（1小时）
📅 后天 19:00-19:30 - 英语作文修改（30分钟）
```

## 🔧 技术实现要点

### 数据模型扩展

```typescript
// 新增：学习档案
interface LearningProfile {
  userId: string;
  totalStudyTime: number;      // 总学习时长
  masteredTopics: Topic[];     // 已掌握知识点
  weeklyStats: WeeklyStat[];   // 周统计
}

// 新增：知识点
interface Topic {
  id: string;
  name: string;
  subject: string;             // 学科
  learnedAt: number;
  relatedTopics: string[];     // 关联知识点
  sourceChatId: string;        // 来源对话
}

// 新增：复习调度
interface ReviewSchedule {
  flashcardId: string;
  nextReviewAt: number;        // 下次复习时间
  level: number;               // SM-2级别
  reviewHistory: ReviewLog[];
}
```
